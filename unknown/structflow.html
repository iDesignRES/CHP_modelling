Detailed Description

Structure to define the parameters of one flow.

Parameters
string"id" = name: Ex. wood
string"def" = definition in database: Ex. spruce_chips
string"cls" = flow class: Ex. hardwood
string"flow_db" = file with flow data in database
string"prop_data" = type of properties: Ex. solid_fuel
Y= mass fraction (kg/kg)
X= molar fraction (mol/mol)
C= molar concentration (mol/l)
phi= volume fraction (m3/m3)
i= atoms vector
j= molecules vector
k= proximate composition vector
ph[3]= phase vector: 1.solid, 2.liquid, 3.gas
string"atom_def" = type of atomic composition: "Y" mass fraction, "X" mol fraction
string"molec_def" = type of molecular composition: "Y" mass fraction, "X" mol fraction
string"prox_def" = type of proximate composition: "Y" mass fraction, "X" mol fraction
